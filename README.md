This project contains tools for extracting, simulating, and analyzing experimental design specifications from the AEA RCT Registry using Large Language Models.

## Documentation

For detailed project documentation, including directory structure, usage workflows, and script details, please determine the [Project Documentation](PROJECT_DOCUMENTATION.md).

## Quick Start

1.  **Install Dependencies**:
    Requires Python 3.8+ and packages listed in individual scripts or notebooks.
    Common requirements: `pandas`, `numpy`, `openai`, `spacy`.

2.  **Explore Data Retrieval**:
    Check `Data Retrival/Design spec/` for the main extraction pipeline.

3.  **Run Analysis**:
    See `text_metrics.ipynb` for text quality evaluation.
